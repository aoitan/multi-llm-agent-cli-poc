# タスク 1.3: A/Bテストスクリプトの雛形作成

## 概要

A/Bテストを実行するためのPythonスクリプトの雛形を`scripts/ab_test_runner.py`として作成する。

## 目的

A/Bテスト機能の基盤を構築し、今後の機能追加を容易にする。

## 受け入れ条件

*   `scripts/`ディレクトリに`ab_test_runner.py`ファイルが存在すること。
*   スクリプトがCLI引数（ユーザープロンプト、実行回数、設定ファイルパス）をパースできること。

## 作業手順

1.  `scripts/`ディレクトリに`ab_test_runner.py`ファイルを作成する。
2.  `argparse`などのライブラリを使用して、CLI引数（ユーザープロンプト、実行回数、設定ファイルパス）をパースする基本的なロジックを実装する。
