# エピック: プロンプト言語A/Bテスト

## 概要

LLMへの内部プロンプトの言語（日本語 vs 英語）が応答品質に与える影響を評価するためのA/Bテストを実施する。LLMのトレーニングデータが主に英語である可能性を考慮し、言語によるパフォーマンスの違いを定量的に測定することを目的とする。
